# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] — 2026-04-19

### Added
- **NueroUI Design System v1.0.0** — initial release
  - `SPEC.md` — Full design specification document (tokens, typography, spacing, radius, shadow, motion, icons, components, platform adaptation)
  - `tokens.css` — Complete CSS design token system with CSS custom properties
  - `nuero-ui.css` — Full component stylesheet (Button, Input, Checkbox, Radio, Switch, Badge, Card, Table, Modal, Toast, Progress, Skeleton, Tabs, Tooltip, Spinner, Navigation, Avatar, Divider)
  - `motion.css` — Motion/animation system (page transitions, micro-interactions, skeleton loading, progress animations, reduced-motion support)
  - `components.html` — Interactive component showcase with theme switcher
  - `platforms/adaptation-guide.md` — Cross-platform adaptation strategy (dark/light/high-contrast/low-power/outdoor/car-HUD/foldable)
  - `icons/` — 120 SVG icons (48 icons × outline + filled variants), linear style, 2px stroke, viewBox 0 0 24 24

### Design System Tokens
- **Colors**: 30+ CSS variables (backgrounds, accents, functional, Chinese traditional accents, text, borders, glows)
- **Typography**: 9 font-size steps, 5 font weights, 3 font stacks (display/body/mono)
- **Spacing**: 4px base grid, 22 spacing steps
- **Radius**: 7 radius steps (0 to full)
- **Shadow**: 6 elevation levels + inner shadow
- **Motion**: 6 duration steps, 5 easing curves

### Icon Set (48 icons)
- Basic: home, search, user, settings, bell, menu, close, check, chevron-*, plus, minus, refresh, download, upload, eye, eye-off, lock, unlock, copy, trash, edit, info, help-circle
- Device: monitor, smartphone, tablet, cpu, wifi, bluetooth, battery, camera, mic, speaker, server, hard-drive
- Communication: send, message, mail, phone, bookmark, share, link, flag, heart, star
- Status: alert-circle, check-circle, x-circle, alert-triangle, loader, zap, activity, trending-up/down, filter, sort, grid, list
